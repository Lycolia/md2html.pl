# Paragraph

インライン要素と分類不能な文字列は全てこのブロックに入る

| 項目   | 値                                                   |
| ------ | ---------------------------------------------------- |
| 子要素 | code, delete, emphasis, image, link, satsuki, strong |

## 例

**入力**

```
ああああ
いいい

**うううえええええ**
```

**出力**

```
<p>ああああ<br />
いいい</p>

<p><strong>うううえええええ</strong></p>
```
