# Header

| 項目   | 値                                                   |
| ------ | ---------------------------------------------------- |
| 子要素 | code, emphasis, delete, image, link, satsuki, strong |

## 例

**入力**

```
# hoge
```

**出力**

```html
<h3 id="hoge">hoge</h3>
```

## 備考

-   `#`の長さ+2のHタグを出す
-   さつき構文のtocで目次が出るようにする
