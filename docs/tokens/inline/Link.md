# Link

| 項目   | 値                                    |
| ------ | ------------------------------------- |
| 子要素 | code, delete, emphasis, image, strong |

## 例

**入力**

```
[hoge](https://example.com)
[![alt-text](https://example.com/1.jpg)](https://example.com)
```

**出力**

```
<a href="https://example.com">hoge</a>
<a href="https://example.com">{Image}</a>
```
